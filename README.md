## Description
Telegram bot for downloading audio with Python

## Techonologies stack
- Python
- aiogram
- pytube

## Requirements
- Python v3+
- bash cli

## Installation

```bash
$ install.sh
```
If you will got permissions denied, so write this command for `install.sh` file:
```bash
$ chmod +x install.sh
```

## Running the app

```bash
$ run.sh
```
